# TGJU Price Crawler

A simple educational web crawler that fetches public price data such as the exchange rate, gold prices, and other market values from **tgju.org**.  
This project is intended for learning purposes only and follows the website's public access rules, including `robots.txt`.

---

## Features
- Fetches daily prices 
- Lightweight and rate-limited to avoid high server load  
- Clean and modular code structure  
- Error handling with retries and timeouts  
- Outputs data to CSV
